Dependancies:

	- ScissDSP ( http://github.com/Sciss/ScissDSP )
	- Scala 2.8 BETA1

To build, copy ScissLib.jar and ScissDSP.jar into the libraries folder. Use included IDEA project file to compile. Use included ant script to create jar.

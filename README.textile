To contact the author, send an email to @contact at sciss.de@. SonogramOverview is based on the @AudioFile@ class of "ScissLib":http://sourceforge.net/projects/scisslib/ and the Constant-Q spectral transform of "ScissDSP":http://github.com/Sciss/ScissDSP .

h3. requirements / installation

Compiles against Scala 2.8.0 and Java 1.6. The repository contains a project file for Intellij IDEA 9 CE, and project files for sbt. Targets for sbt:

* @clean@
* @compile@
* @doc@
* @package@
* @run@

Run an initial @sbt update@ to install the depenancies. ScalaAudioFile must have been previously locally published using @sbt publish-local@ from its main folder.
